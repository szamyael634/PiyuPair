Note:
Organizations are the handlers of the tutors
Organization handles, Screening, approval of tutors and learning materials
The System collects 5% commission on every successful session of Student - Tutor
The System will provide a test or quiz maker.
The System will have a Google classroom like feature where activities are to be posted by tutor and outputs to be submitted by student
Every Organization will generate a Unique Code for applying tutors to use in the registration
The System will have a Chat System( private messages, and group chats)
The System will have a Rating system where Tutors and organizations can be rated by stars, additional rates will be added if there are proofs of the improvement of grades because of the tutoring.
the System will have a automatic screening of tutor applicants like, screening of Credentials(Grades, Trainings, Certificate) the same goes to the learning material (checks whether the learning material submitted is relevant to the topic to be discussed)
Every approved submitted material will be stored in each organization's library for future purposes.
The system also has a study group feature where a student can form a study group where other students can join(becomes a groupchat or forum where students message each other, send files, etc.)
there will be a newsfeed for all (Like Facebook) where posts like asking academic questions, posting of upcoming webinars, seminars
The system will also have a Open Library where Organizations will share Learning Materials that will be open to all students
The System will also have a payment channel
In order for the system to collect commission fees, student will book the tutor for a session(a book form will be present asking, number of hours, the topic to be discussed etc.)

Registration Process

Tutor

applicant will request a unique code to the organization (either chat or meet them face to face), upon getting the unique code, applicant will proceed with the account registration. applicant will be required to submit personal information, credentials to prove he/she is qualified to teach ( Grades, Trainings, Certifications) then the system will check whether the applicant is qualified or not, if not then automatic reject, if pass then application will be checked by the organization. if organization approves, then account is created

Student

Student needs to provide personal information, educational information (Program, student ID no., year level, Department)
students will be approved by the admin

Organization

a request to create an organization account needs to inform the admin, then admin will create a organization account
